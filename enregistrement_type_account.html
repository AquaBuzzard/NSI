Type de compte

Chez nous vous avez la possibilité de créer deux types de compte differents
lisez bien l'information sur chaque type de compte ci-dessous puisque chaque type de compte
est utilisé de manière totalement différente.

Compte Client

Ce type de compte est comme son nom l'indique utilisé par les clients pour les
deliverys et pour échanger des Chickenpoints par des produits d'edition limitée

Avec ce type de compte vous pouvez:


  1. Commander de la nourriture à domicile et gagner des Chickenpoints
  2. Gérer et échanger vos Chickenpoints
  3. Béneficier des coupons et des promotions exlusives pour les clients avec compte

Compte Professionnel

Ce type de compte est pour les personnes qui veulent candidater pour un poste de travail chez nous.

Attention!!! Ce type compte n'a pas d'accès à notre environnement de travail, il est reservé pour les candidatures, une fois embauché/e vous recevrez accès à l'environnement interne (ou pas, en fonction de votre poste).

avec ce type de compte vous pouvez:

  1. Déposer une candidature d'embauchement chez nous.
  2. Verifier l'état de votre candidature
  3. Verifier les horaires de vos prochains entretiens

Maintenant que vous-êtes informé, vous pouvez continuer et créer votre compte.
