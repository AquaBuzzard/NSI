Type de compte

Chez nous vous avez la possibilité de créer deux types de compte differents
lissez vien l'information sur chaque type de compte qui est ci-dessous puisque chaque type de compte
est utilisée d'une manière totalement differente.

Compte Client

Ce type compte est comme son nom l'indiaue utilisé par les clients pour les
deliverys et pour échanger des Chickenpoints par des produits d'edition limité

Avec ce type de compte vous pouvez:


  1. Commander de la nourriture à domicile et gagner des Chickenpoints
  2. Gérer et échanger votres Chickenpoints
  3. Béneficier des coupons et des promotions exlusives pour les clients avec compte

Compte Professionel

Ce type de compte est pour les personnes aui veulent candidater pour un poste de travaille chez nous.

Attention!!! ce type compte n'a pas d'access à notre environement de travaille il est reservé pour les candidatures, une fois embauché/e vous recevrez accès à l'environement interne (ou pas en fonction de votre poste).

avec ce type de compte vous pouvez:

  1. Deposser une candidature d'embauchement chez nous.
  2. Verifier l'état de votre candidature
  3. Verifier les horaires des vos prochaines interviews

Maintenant que vous-êtes informez vous pouvez continuer et créer votre compte.
